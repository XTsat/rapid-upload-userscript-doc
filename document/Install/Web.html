网页版无需安装插件, 点开即用, 推荐无法安装插件的用户使用

# 常见问题

  1. 如何转存秒传链接？

    填入 秒传链接、转存路径，点击'解析链接并转存'。 显示 "errno":0 即为转存成功。

  2. 如何生成秒传链接？

    选择文件后，等待秒传链接出现。 大文件计算速度较慢，请耐心等待。 需要任意账号的网盘中存有至少一份文件才能保证秒传链接的有效性!
